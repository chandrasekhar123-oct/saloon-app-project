# Saloon Essy - User App

Saloon Essy is a premium React Native application built with **Expo**, designed to provide a seamless salon booking experience for users. The app features a modern, clean interface with a focus on ease of use and visual appeal.

## 🚀 Features

### 1. Authentication & Onboarding
- **Splash Screen**: Professional onboarding with brand identity.
- **OTP Login**: Secure phone number-based authentication.
- **Social Integration**: Option for Google Sign-in.

### 2. Discovery (Home)
- **Location Services**: Real-time location tracking (simulation).
- **Search & Filters**: Find salons by name, service, or category.
- **Promotional Banners**: Stay updated with the latest offers and discounts.
- **Service Categories**: Quick browsing for Haircuts, Beard Styling, Facials, Spa, and more.

### 3. Booking Experience
- **Salon Details**: Comprehensive info including addresses, ratings, and operational status.
- **Multi-Service Selection**: Add multiple services to a single booking.
- **Stylist/Worker Selection**: Choose your preferred professional based on experience and rating.
- **Slot Management**: Pick a convenient time slot from available options.
- **Booking Summary**: Review your selection and select payment methods (UPI, Card, Cash).

### 4. User Management
- **My Bookings**: Keep track of upcoming, completed, and cancelled appointments.
- **Favorites**: Save your preferred salons for quick access.
- **User Profile**: Manage personal details and app settings.
- **Reviews & Ratings**: Share your experience by rating stylists and uploading photos.

## 🛠️ Tech Stack

- **Framework**: [React Native](https://reactnative.dev/) with [Expo](https://expo.dev/)
- **Icons**: [Expo Vector Icons](https://docs.expo.dev/guides/icons/) (Ionicons, MaterialIcons, Feather)
- **Styling**: Vanilla StyleSheet with a custom modern design system.
- **Assets**: Optimized images and icons for a premium feel.

## 🎨 Design System

- **Primary Color**: `#E23744` (Vibrant Red)
- **Background**: `#F8F9FB` (Soft Grey/White)
- **Typography**: Focused on readability and hierarchy.
- **UI Patterns**: Glassmorphism touches, soft shadows, and rounded interactive elements.

## 📦 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run the App**:
   ```bash
   npx expo start
   ```

3. **Open on Device**:
   - Scan the QR code using the **Expo Go** app (Android/iOS).

## 📂 Project Structure

- `App.js`: Main application logic and navigation.
- `assets/`: Image resources and icons.
- `app.json`: Expo configuration.
- `package.json`: Project dependencies and scripts.
